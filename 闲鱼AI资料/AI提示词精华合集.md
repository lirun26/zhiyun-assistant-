# 第四部分：AI写作万能公式

## 4.1 爆款标题公式
```
公式一：数字+悬念+利益
公式二：身份+痛点+方案
公式三：反常识+好奇+确认

示例：
- "3个技巧，让你的效率提升300%"
- "为什么你写的文案没人看？"
- "同事不会告诉你的5个工作秘密"
```

## 4.2 开头写法
```
公式一：痛点切入
公式二：数据吸引
公式三：故事引入
公式四：名言开头

示例：
"你是不是也有这样的困扰：每天忙得脚不沾地，却总觉得没做出什么成果？"
"据统计，90%的人都踩过这个坑..."
"我曾经也是一个只会埋头苦干的人，直到..."
```

## 4.3 结构模板

### 清单体
```
写好清单体的关键：
1. 数字开头，如"5个方法"
2. 每个观点2-3句话
3. 结尾有总结或升华

格式：
[数字] + [核心观点] + [解释/案例]

示例：
1. 先完成再完美
   很多人死在追求完美上，其实先跑通流程比什么都重要
```

### 故事体
```
故事体万能结构：
1. 冲突：遇到了什么问题
2. 努力：做了哪些尝试
3. 结果：最终怎么样了
4. 感悟：从中学到了什么

示例：
"我曾经连续加班一个月，项目还是延期了...
后来我用了一个方法，三天就完成了..."
```

# 附录：使用技巧

## 如何提问获得更好答案

1. **具体描述背景**：不要只说"帮我写文案"，要说明"我是卖XX的卖家，需要在闲鱼上发布"
2. **说明使用场景**：让AI知道内容给谁看
3. **指出限制条件**：如字数、风格、格式
4. **要求多版本输出**：生成后可选择、修改
5. **追问优化**：对不满意的地方说"这个角度不好，换一个"

## 常见场景问答

### 问：AI回答太泛泛怎么办？
答：加约束条件，如"用具体案例说明"、"给出可操作的步骤"

### 问：AI理解有偏差怎么办？
答：补充背景信息，明确说"不是这个意思，是XX"

### 问：想要创意但AI太保守？
答：加一句"请大开脑洞"、"打破常规"
