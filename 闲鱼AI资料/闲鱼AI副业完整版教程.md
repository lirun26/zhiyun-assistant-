## 2.2 养号7天完整流程

**为什么养号？**

新账号直接发布商品容易被判定为营销号，导致曝光低、容易被封。

**养号核心原则：**
- 模拟真实用户行为
- 让平台认为你是正常用户
- 建立账号权重

**Day 1-2：完善资料，熟悉平台**

任务：
- [ ] 完善个人资料（头像、昵称、简介、背景）
- [ ] 逛闲鱼首页，刷商品至少30分钟
- [ ] 关注5-10个同行账号
- [ ] 点赞20个商品
- [ ] 评论10个商品（正常评论，如"看起来不错"）

**Day 3-4：开始互动，培养活跃度**

任务：
- [ ] 签到，领取闲鱼币
- [ ] 每天浏览商品至少30分钟
- [ ] 收藏10个同行爆款商品
- [ ] 在同行商品下评论互动
- [ ] 尝试在闲鱼搜索自己想做的品类

**Day 5-6：模拟购买，建立信用**

任务：
- [ ] 买1-2个性价比高的小东西（几元到十几元）
- [ ] 收到货后及时确认收货
- [ ] 给卖家5星好评
- [ ] 开始有购买记录，更真实

推荐购买：
- 虚拟资料试水（如几元的壁纸、表情包）
- 便宜的小工具

**Day 7：发布测试商品**

任务：
- [ ] 发布1-2个测试商品
- [ ] 不要急于上架大量商品
- [ ] 观察商品曝光情况

# 第三章：选品篇

## 3.1 选品的5个核心标准

**标准一：需求真实存在**

判断方法：
- 搜索关键词，看看有多少人在卖
- 看看同行销量怎么样
- 看看评论区买家在问什么

标准：
- 有3个以上同行在卖且有销量 → 需求真实
- 评论区很多人问但没人回答 → 需求迫切

**标准二：竞争适度**

判断方法：
- 搜索结果50-200个同行 → 竞争适中
- 搜索结果<50个 → 蓝海市场
- 搜索结果>500个 → 红海市场

标准：
- 不要太红（竞争激烈）
- 不要太蓝（没有需求）
- 50-200个同行最佳

**标准三：利润空间**

判断方法：
- 成本：几元到几十元
- 售价：9.9元到99元
- 利润：至少10元以上

标准：
- 成本<售价×30%
- 单笔利润>10元
- 可以自动化发货

**标准四：可持续销售**

判断方法：
- 是长期需求还是短期热点
- 有没有季节性
- 能不能持续更新

标准：
- 避开季节性太强的（如期末资料）
- 选择长期需求品类
- 能持续更新内容

**标准五：自己hold住**

判断方法：
- 你对这个领域了解多少
- 能不能回答买家问题
- 有没有持续更新能力

标准：
- 选择自己熟悉的领域
- 能提供售后支持
- 有持续更新能力
