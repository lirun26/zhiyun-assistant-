# 📚 小说辅助助手 (novel-assistant)

参考笔灵AI功能，统一协调所有小说创作相关技能的一站式创作平台。

## 🚀 快速开始

```
你说：帮我写一个都市商战小说大纲
我就会：
1. 调用 novel-generator 生成完整大纲
2. 输出：世界观 + 人物设定 + 分章大纲
```

## 📖 功能列表

| 功能 | 说明 |
|:---|:---|
| 📋 大纲生成 | 根据题材自动生成完整大纲 |
| ✍️ 章节创作 | 6步规范流程创作单章节 |
| 🎯 黄金开头 | 生成吸引编辑的开头 |
| 🔍 素材查询 | 背景/设定/角色素材库 |
| 📝 续写扩写 | 润色扩展已有内容 |
| 🔄 多模型审稿 | 三路并行审阅 |
| 📮 投稿指南 | 10+平台投稿攻略 |
| ✅ 质量检查 | P0/P1红线 + 评分系统 |

## 🔧 安装

技能已安装，直接使用即可。

## 📝 使用规范

### P0红线（绝对禁止）
- ❌ AI词汇：赋能、进化、迭代、闭环、赛道
- ❌ 感悟式结尾
- ❌ 感叹式结尾
- ❌ 上帝视角
- ❌ 超过3000字

### 评分标准
- 90-100：精品
- 85-89：优秀
- 75-84：良好
- <75：需修改

## 📚 关联技能

- `novel-generator` - 大纲生成
- `open-novel-writing` - 章节创作
- `novel-workshop` - 多模型审稿
- `story-cog` - 故事创意
- `content-writer` - 内容续写
