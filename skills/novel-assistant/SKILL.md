# 📚 小说辅助助手

参考笔灵AI功能，统一协调所有小说创作技能，提供完整的小说创作流程支持。

## 🎯 核心功能

| 功能 | 调用技能 | 说明 |
|:---|:---|:---|
| 大纲生成 | novel-generator | 根据题材/关键词生成完整大纲 |
| 章节创作 | open-novel-writing | 按规范创作单章节 |
| 黄金开头 | novel-generator | 生成吸引人的开头 |
| 素材库 | 内置素材库 | 提供背景、设定、角色素材 |
| 续写/扩写 | content-writer | 润色和扩展已有内容 |
| 多模型审稿 | novel-workshop | 三路并行审阅（逻辑/文学/锐评） |
| 投稿指南 | 内置指南 | 10+平台投稿攻略 |
| 质量检查 | 内置审核 | P0/P1红线检查，评分≥85 |

## 📖 大纲生成

**输入**：题材、风格、关键词、主角设定

**输出**：完整大纲，包含：
- 世界观设定
- 主角/配角人设
- 主线剧情（起承转合）
- 分章大纲（10-20章）
- 伏笔埋设
- 爽点设计

**调用技能**：`novel-generator`

```bash
# 使用 novel-generator 技能生成大纲
# 参考：skills/novel-generator/SKILL.md
```

## ✍️ 章节创作（6步流程）

| 步骤 | 内容 | 可跳过 |
|:---|:---|:---|
| 1 | 创建章节规格文档（Spec） | ❌ |
| 2 | 创作正文 | ❌ |
| 3 | 创建评审报告 | ❌ |
| 4 | P0/P1红线检查 | ❌ |
| 5 | 修正问题 | ❌ |
| 6 | 发送/存档 | ❌ |

### 🔴 P0红线（绝对禁止）
- AI词汇（赋能、进化、迭代、闭环、赛道）
- 感悟式结尾
- 感叹式结尾（"！"连续出现）
- 上帝视角
- "第X部分"标题
- 分隔线
- 超过3000字

### 🟡 P1建议避免
- 套路化表达
- 流水账描写
- 抽象心理描写
- "冷笑"、"侥幸"等重复词

### 📊 评分标准
| 分数 | 评价 | 动作 |
|:---|:---|:---|
| 90-100 | 精品 | 可直接发布 |
| 85-89 | 优秀 | 小改可发布 |
| 75-84 | 良好 | 需修改 |
| 60-74 | 合格 | 大改 |
| <60 | 不合格 | 重写 |

## 🎯 黄金开头公式

```
1. 冲突开场（30%）
   主角陷入困境/危机，立刻制造紧张感
   
2. 异能展示（30%）
   开场就展示主角独特能力/身份
   
3. 悬念设置（25%）
   提出问题，吸引继续阅读
   
4. 情绪代入（15%）
   引发读者情感共鸣
```

## 📝 使用示例

### 示例1：生成大纲
```
用户：我想写一个都市商战小说，主角是风水师重生
助手：
1. 分析需求 → 都市重生+风水玄学+商战
2. 调用 novel-generator 生成完整大纲

助手：

2. 按6步流程创作

```
