# 🎯 黄金开头模板

## 公式一：冲突开场（最推荐）

```
主角陷入困境/危机，开场就制造紧张感
```

**模板**：
```
"先生，您的公司......三个月内必倒！"

林渊看着眼前这个西装革履的中年男人，嘴角勾起一抹冷笑。

他没说自己是重生者，也没说对方请的风水师其实是个骗子。

只是轻轻抬了抬眼皮："让你背后那位来跟我谈。"
```

**要点**：
- 第一句就是冲突/危机
- 立刻展示主角能力
- 设置悬念

---

## 公式二：异能展示

```
开场就展示主角独特能力/身份
```

**模板**：
```
林渊睁开眼的那一刻，整个办公室的风水格局在他眼中纤毫毕现。

龙脉走势、财气聚散、小人方位......

他嘴角微微上扬。

"这栋楼，三天后要出事。"

身后的秘书脸色惨白："林......林总，您怎么知道......"

他是林渊。

曾经站在风水界顶端的男人，如今重生归来。
```

---

## 公式三：悬念设置

```
提出问题，吸引继续阅读
```

**模板**：
```
母亲临终前说的那句话，林渊记了二十年。

"你不是我的亲生儿子。"

他是京城林家唯一的继承人，怎么可能不是亲生的？

直到他被养子夺走一切，在街头冻饿而死。

直到他重生回到二十岁那年。

他才明白这句话的真正含义。
```

---

## 公式四：情绪代入

```
引发读者情感共鸣
```

**模板**：
```
三百万。

这是林渊被净身出户后，账户里最后的余额。

前世的今天，是他被赶出林氏集团的日子。

养子林轩带着私生子身份夺走了他的一切——家业、地位、甚至他的未婚妻。

而他只能在出租屋里，孤独地死去。

但现在，一切都不同了。

因为他重生在了命运转折的前一天。
```

---

## ❌ 禁止使用的开头

1. 景物描写开头（"月光洒在窗台上......"）
2. 大量背景介绍（"林渊是xxx人，他从小......"）
3. 平淡日常（"今天林渊去上班了......"）
4. AI腔（"赋能"、"进化"、"赛道"）
