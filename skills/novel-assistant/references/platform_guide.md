## 番茄小说

**特点**：流量最大，新人容易出头

**投稿方式**：
1. 电脑端：https://fanqie.cn
2. 创建作品 → 填写大纲 → 开始创作
3. 满2万字可申请签约

**攻略**：
- 前3章至关重要，决定是否推流量
- 黄金开头 + 第一个高潮要在前5章
- 日更4000以上可获得更多推荐

**适合题材**：都市爽文、重生、系统流、甜宠

---

## 起点中文网

**特点**：付费阅读天花板，作者圣地

**投稿方式**：
1. 电脑端：https://www.qidian.com
2. 联系编辑投稿，或内投大纲

**攻略**：
- 精品化路线，文笔要求高
- 编辑看大纲，不只看开头
- 建议先在其它平台练手

**适合题材**：玄幻、仙侠、军事、历史

---

## 晋江文学城

**特点**：女性向绝对霸主

**投稿方式**：
1. 电脑端：https://www.jjwxc.net
2. 满2万字可申请签约

**攻略**：
- 言情、纯爱分类最强
- 文笔细腻，情感描写要求高
- 申榜需要收藏达标

**适合题材**：言情、耽美、百合、甜文

---

## 七猫中文网

**特点**：免费阅读，保底分成，新人友好

**投稿方式**：
1. 电脑端：https://www.qimao.com
2. 直投编辑邮箱

**攻略**：
- 保底价格高，风险低
- 文风偏新媒体
- 编辑回复快

**适合题材**：都市、总裁、甜宠、虐恋

---

## 飞卢小说网

**特点**：更新为王，日更一万起步

**投稿方式**：
1. 电脑端：https://www.faloo.com
2. 创建作品即可发文

**攻略**：
- 更新慢等于死刑
- 脑洞文、新媒体文吃香
- 追读数据决定推荐

**适合题材**：脑洞文、系统流、直播文、游戏文

## 投稿邮件格式

**收件人**：各平台官方投稿邮箱

**邮件标题**：
```
【投稿】《书名》+ 题材 + 作者名
```

**邮件正文**：
```
书名：
笔名：
联系方式：
作品类型：（都市/玄幻/言情等）
预计字数：
内容简介：（200-500字）
大纲：（500-1000字）
前5章正文：
```

## ⚠️ 注意事项

1. **不要一稿多投** - 等编辑回复后再投其他平台
2. **保留底稿** - 投稿前务必备份
3. **看清合同** - 签约前仔细阅读合同条款
4. **防骗** - 不收取任何费用的平台才是正规平台
