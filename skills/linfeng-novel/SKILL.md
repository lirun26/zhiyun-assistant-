# 📋 阿里云 OpenClaw 小说创作规范 — 番茄拒稿原因与整改任务

**执行角色**：阿里云 OpenClaw（agent:main:aliyun）
**目标**：重开新书，确保番茄签约通过

---

## 一、番茄小说平台签约核心要求（最低门槛）

1. **字数**：每章 **2000-3500 字**（低于2000字直接拒）
2. **节奏**：前三章必须出现第一个冲突/爽点/钩子，不能写流水账
3. **开篇**：第1章第1句就要有事件/冲突，不能从环境描写开始
4. **无CP/单CP**：末世文可以无CP但难度极高，建议设定1V1感情线
5. **逻辑自洽**：资金、物品、人物关系前后一致，番茄编辑会逐字检查
6. **原创性**：AI写作特征太明显会被识别拒稿（详见下方"AI味识别清单"）

## 二、番茄拒稿原因归纳（原书《重生末世废土之生存法则》）

### ❌ 致命问题（P0 — 直接拒稿）

#### 1. 字数严重不达标
| 章节 | 实际字数 | 番茄要求 | 差距 |
|------|---------|---------|------|
| 第9章 | ~1459字 | 2000字+ | ❌ 不足40% |
| 第10章 | ~1150字 | 2000字+ | ❌ 不足60% |
| 第11章 | ~1193字 | 2000字+ | ❌ 不足60% |
| 第12章 | ~1909字 | 2000字+ | ❌ 差100字 |

**后果**：连续5章字数不达标，番茄编辑直接判定"章节过短，水分过多，拒稿"

#### 2. 开篇前三章无有效冲突
第1章用了大量篇幅讲前世背景（深冬→背叛→惨死→重生→金手指→立誓），信息密度低。番茄爆款开篇公式：
```
第1句：事件冲突（不是氛围）
第3段：主角目标亮相
第1页：第一个障碍/危机出现
```

#### 3. AI写作特征太明显（番茄有AI检测）
- 大量内心独白替代动作描写
- "绝非...绝不...绝不再..."高频否定排比
- 固定句式："苏清鸢的嘴角勾起一抹..."
- 每章结尾有"本章总结"和"下章预告"（AI才会写这种东西）
- "末世爆发前XX天"倒计时句式反复出现

#### 4. 平台定位偏差（赛道选错）
- 《重生末世废土》属于**末世独立女主向**，番茄冷门赛道
- 番茄主流是**都市言情/甜宠/1V1**，哪怕末世文也要有感情线
- **建议**：要么改平台（起点/七猫），要么改成**有1V1恋爱线的末世文**

---

### ❌ 逻辑硬伤（被编辑逐字检查发现）

#### 5. 资金逻辑矛盾
- **问题**：苏清鸢现金15.5万，但支出43万，缺口27.5万
- **现状**：ERRORS.md记录了，但未修复
- **影响**：编辑发现直接拒稿

#### 6. 物品来源不明
- 电棍、匕首等武器在采购清单中没有出现就使用了
- 每件物品必须"先出现，再使用"

#### 7. 交通方式前后矛盾
- 第9章骑摩托车，第12章写成"开车"，矛盾

---

### ⚠️ 次要问题（影响评分）

#### 8. 章节末尾"总结+预告"必须删除
第12章末尾有"本章总结"和"第一卷高潮预告"，这是AI写作标志，番茄编辑会识别

#### 9. 女主性格靠"念"不靠"做"  
用大段独白宣告性格转变，而不是通过行动展现

#### 10. 任务模板字数要求与番茄不匹配
原模板写"1500-2500字/章"，番茄要求"2000-3500字"

## 三、新书创作规范（固化要求）

### 📌 核心规范（每次创作前必须读取）

```
项目路径：/home/admin/.openclaw/workspace/novels/{书名}/
设定文件：/home/admin/.openclaw/workspace/novels/{书名}/.learnings/
输出目录：/home/admin/.openclaw/workspace/novels/{书名}/output/
```

### ✍️ 章节字数标准
- **每章 2500-3500 字**（上限3500，禁止超过）
- 字数低于2000字 → 必须补内容，禁止发布
- 字数超过3500字 → 必须拆分章节

### 🎯 开篇公式（前三章必须做到）
```
第1章：
  第1句：事件驱动（不是氛围描写）
  前3段：主角身份+处境明朗
  前1页：第一个冲突/危机出现
  前3页：金手指/独特能力亮相

第2章：
  主角用能力解决第一个小问题（验证金手指）
  埋1-2个后续伏笔

第3章：
  出现第一个反派/障碍
  建立本章核心冲突
```

### 🚫 AI味识别清单（每章发布前自检）
- [ ] 无"本章总结"/"下章预告"段落
- [ ] 无"绝非...绝不...绝不再..."句式
- [ ] 无连续3段以上内心独白
- [ ] 主角性格通过行动展示，不是独白宣告
- [ ] 每件物品"先出现再使用"
- [ ] 固定句式（如"嘴角勾起一抹..."）出现不超过2次/章

### 🔢 番茄签约字数节奏
```
投稿要求：连续5-10章每章2000字+才能申请签约
建议节奏：前30章每章2500-3000字，快速积累可投稿量
签约节点：50章、100章各申请一次
```

### 📊 发送前检查清单（每章必须过一遍）
```
[ ] 字数确认（工具统计，确认2500-3500字）
[ ] 读上章末尾5段，时间/地点/人物无断裂
[ ] 开篇500字有事件冲突，不是氛围描写
[ ] AI味自检全部通过
[ ] 无资金/物品/交通等逻辑矛盾
[ ] 无新人物凭空出现（必须有名字+身份+初次登场描写）
[ ] 无时间线矛盾
[ ] 章节末尾无"总结"和"预告"
[ ] PLOT_POINTS.md已更新
[ ] CHARACTERS.md已更新（新角色）
```

### 📁 配套文档维护（每次写完章节立即更新）
```
每次写完章节后必须执行：
1. 更新 PLOT_POINTS.md → 勾选[x]已完成情节，补充[ ]新伏笔
2. 更新 CHARACTERS.md → 新角色立即补充条目
3. 更新 LOCATIONS.md → 新地点立即补充
4. 更新 ERRORS.md → 如有逻辑问题立即记录（哪怕是medium级）
```
