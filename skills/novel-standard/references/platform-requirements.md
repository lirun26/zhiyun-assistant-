# 各平台签约要求对比

## 番茄小说

| 项目 | 要求 |
|-----|------|
| **签约字数** | 连续5-10章2000字+可申请签约 |
| **每章字数** | 2000-3500字（低于2000直接拒） |
| **开篇** | 前三章必须有冲突/爽点/钩子 |
| **节奏** | 不能流水账，必须有起伏 |
| **感情线** | 无CP/单CP极难签约，建议1V1 |
| **AI检测** | 有，会识别AI写作特征 |
| **冷门赛道** | 末世独立女主属于冷门，建议加感情线或换平台 |

## 七猫小说

| 项目 | 要求 |
|-----|------|
| **签约方式** | 免费签约，无门槛 |
| **字数** | 每章2000字+ |
| **推荐机制** | 完读率为主 |
| **无CP接受度** | ✅ 可接受 |
| **适合类型** | 末世、种田、重生 |

## 起点女频

| 项目 | 要求 |
|-----|------|
| **签约门槛** | 较高，需10万字+ |
| **每章字数** | 2000-3000字 |
| **无CP接受度** | ✅ 可接受 |
| **适合类型** | 末世异能、独立女主 |
| **特点** | 老牌平台，读者粘性高 |

## 飞卢

| 项目 | 要求 |
|-----|------|
| **签约方式** | 快速签约 |
| **每章字数** | 2000-3000字 |
| **日更要求** | 高，需日更3000字+ |
| **无CP接受度** | ✅ 可接受 |
| **适合类型** | 末世、系统、爽文 |

## 平台选择建议

```
冷门赛道（末世独立女主）优先级：
1. 七猫小说 — 签约简单，免费，无CP可接受
2. 起点女频 — 适合有深度剧情的末世文
3. 飞卢 — 适合快节奏系统文
4. 番茄小说 — 难度最高，除非加感情线
```

## 快速选择决策树

```
是否有感情线？
├── 有1V1感情线 → 番茄/起点/七猫均可
└── 无感情线（独立女主）
    ├── 想快速签约 → 七猫
    ├── 想写深度剧情 → 起点女频
    └── 纯爽文快节奏 → 飞卢
```
