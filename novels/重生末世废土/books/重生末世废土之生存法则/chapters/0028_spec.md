# 第28章 规格 (Spec)

## 📝 章节信息
- **章节编号**: 28
- **标题**: 故人重逢
- **前情回顾**: 第27章结尾，仇人（周红/小红等四人）将从第七避难所来翡翠湾投靠
- **目标字数**: 3500字
- **时间线**: 末世第26天上午 → 下午

## 🎯 本章核心事件
1. 仇人正式登场（四人：周红、赵梅、王美、李强）
2. 苏清鸢与仇人正面交锋，内心博弈
3. 小红的表演 vs 苏清鸢的冷静观察
4. 身份初步确认，但不打草惊蛇

## 📖 剧情大纲

### 第一部分：仇人到来（上午）
- 第七避难所的车到达
- 四人下车：三女一男
- 苏清鸢带队迎接，表面热情，内心警惕
- 观察：周红的红头发、小动作、说话方式

### 第二部分：初次对话（上午）
- 自我介绍：小红（周红）、赵梅、王美、李强
- 周红表演：装可怜、卖惨、示好
- 苏清鸢配合：表面接受，内心记录
- 观察其他三人的反应

### 第三部分：安置与监视（中午）
- 安排住处：与原住民分开
- 苏清鸢布置监视：陈刚、林雨暗中观察
- 周红试探安全屋情报
- 苏清鸢应对：只透露表面信息

### 第四部分：暗中确认（下午）
- 陈刚汇报观察结果
- 确认：周红就是前世凶手
- 其他人不知情，只是跟班
- 苏清鸢决定：继续观察，等待时机

## 👥 仇人信息

### 周红（小红）
- 身份：红发女人，前世杀死苏清鸢的凶手
- 外貌：红头发，蝴蝶结发夹
- 性格：残忍、狡诈、善于表演
- 目的：不明，可能想找长期庇护所

### 赵梅
- 身份：短发瘦小女人
- 外貌：短发，身材瘦小
- 性格：胆小，附和周红
- 角色：跟班

### 王美
- 身份：同伙
- 外貌：普通
- 性格：沉默，服从
- 角色：跟班

### 李强
- 身份：唯一的男人
- 外貌：高个子，有点结巴
- 性格：老实，被胁迫
- 角色：可能是被周红等人控制的

## 🔧 关键数据

### 翡翠湾团队
- 16人（含苏清鸢）
- 物资：够25天
- 弹药：弩箭13支，燃烧瓶8个

### 环境状态
- 气温：零下10度
- 天气：阴天，有雨
- 丧尸活跃度：持续上升

## ⚠️ 注意事项

### 爽点设计
1. **信息差**：苏清鸢知道她们是谁，她们不知道
2. **表演vs观察**：小红在表演，苏清鸢在记录
3. **心理博弈**：小红的试探 vs 苏清鸢的应对
4. **细节暗示**：苏清鸢注意到的小细节

### 写作要点
- 每段第一句不超过15字
- 对话不用"冷笑"、"侥幸"、"淡淡道"
- 动作具体化
- 心理通过动作表现，不直接写
- 章尾留钩子：苏清鸢的复仇计划开始

### 逻辑一致性
- 时间：末世第26天上午→下午
- 团队：16人
- 物资：延续前文数据
