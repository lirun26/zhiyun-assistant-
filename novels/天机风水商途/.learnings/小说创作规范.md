## 🔥 创作前必检（4项）

### ① 确认时间线
```bash
grep -n "二十年前\|三年前\|二十年前" .learnings/CHARACTERS.md
```
- ❌ 禁止写"三年前灭门"（正确的是二十年前）
- ❌ 禁止写"觉醒三年"（才一个半月）
- ❌ 禁止写"三年前收徒"（时间不够）

### ② 确认角色关系
```bash
cat .learnings/CHARACTERS.md
```
- 新角色出场前必须在CHARACTERS.md登记
- 师徒/盟友/敌人关系必须明确
- 角色年龄/身份必须合理

### ③ 确认衔接
```bash
tail -10 第XX章.md
```
- 本章开头必须承接上章结尾
- 确认角色在同一地点
- 确认角色状态连续

### ④ 确认规范
- 都市风水商战（非修仙）
- 用词：道行/真气/气场（非修为/灵力）
- 每章≥4000字

## 🔥 写完后必检（6项）

### ① 字数
```bash
wc -m 第XX章.md
```
- 必须 ≥4000字

### ② 修仙词汇
```bash
grep -n "筑基\|金丹\|元婴\|化神\|修为\|修仙\|灵气\|炼气" 第XX章.md
```
- 必须无修仙词汇

### ③ AI词汇
```bash
grep -n "赋能\|维度\|赛道\|迭代" 第XX章.md
```
- 必须无AI词汇

### ④ 时间线
```bash
grep -n "年前\|年后\|个月\|时日" 第XX章.md
```
- 确认无时间线矛盾

### ⑤ 结尾
```bash
tail -5 第XX章.md
```
- 确认结尾不重复（检查前3章结尾）
- 禁止感悟式/感叹式结尾

### ⑥ 金句/爽点
- 人工检查：是否有打脸爽点
- 人工检查：是否有金句
