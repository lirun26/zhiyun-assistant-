# 风水知识库 STORY_BIBLE

## 一、八宅派核心体系

### 1.1 基础理论

八宅派是风水学中最重要的理气派别之一，以《周易》为理论基础。

**核心概念**：
- 以八卦（乾、坎、艮、震、巽、离、坤、兑）为基础
- 将住宅分为东四宅（坎、离、震、巽）与西四宅（乾、坤、艮、兑）
- 以人的生年八卦属性与住宅八卦配合论吉凶

### 1.2 大游年歌诀（核心工具）

八宅派以大游年歌诀为主要工具，判断阳宅八方吉凶。

**歌诀全文**：
```
乾六天五祸绝延生
坎五天生延绝祸六
艮六绝祸生延天五
震延生祸绝五天六
巽天五六祸生绝延
离六五绝延祸生天
坤天延绝生祸五六
兑生祸延绝六五天
```

**解读**：
- 第一个字代表"伏位"（起点）
- 顺时针排列其他七位

**八宫卦位对应**：
| 卦 | 五行 | 方位 | 吉凶 |
|:---:|:---:|:---:|:---:|
| 乾 | 金 | 西北 | 伏位 |
| 坤 | 土 | 西南 | 伏位 |
| 艮 | 土 | 东北 | 伏位 |
| 兑 | 金 | 西 | 伏位 |
| 坎 | 水 | 北 | 伏位 |
| 离 | 火 | 南 | 伏位 |
| 震 | 木 | 东 | 伏位 |
| 巽 | 木 | 东南 | 伏位 |

### 1.3 九星吉凶

**四吉星**：
| 星名 | 主吉 | 宜 |
|:---:|:---:|:---|
| 生气 | 桃花、财运 | 开门、卧床 |
| 延年 | 长寿、健康 | 书房、老人房 |
| 天医 | 疾病康复 | 药房、病床 |

**四凶星**：
| 星名 | 主凶 | 忌 |
|:---:|:---:|:---|
| 五鬼 | 火灾、疾病 | 厨房、卧室 |
| 祸害 | 官非、口舌 | 客厅 |
| 绝命 | 早亡、绝嗣 | 任何位置 |
| 六煞 | 破财、桃花 | 门路 |

### 1.4 东四宅与西四宅

**东四宅**：坎、离、震、巽
**西四宅**：乾、坤、艮、兑

**吉凶配合原则**：
- 东四命居东四宅 → 大吉
- 西四命居西四宅 → 大吉
- 东西相混 → 不吉

### 1.5 门主灶关系

八宅看阳宅，主要看三个元素：
1. **门**（气口）：最重要，决定纳气
2. **主**（主卧）：主人睡眠位置
3. **灶**（厨房）：饮食制作之地

**原则**：
- 大门宜开在四吉方
- 主卧宜在四吉方
- 灶台宜在四凶方（以凶制凶）

## 三、商业风水应用

### 3.1 店铺/办公室财位

**明财位**（固定位置）：
- 大门斜对角
- 宜亮不宜暗
- 宜坐不宜卧

**暗财位**（流年变化）：
- 根据九宫飞星每年变化
- 2024年财位：正北、正南

### 3.2 商铺选址

**好商铺条件**：
1. 门面宽敞，招牌醒目
2. 门前无大树、电线杆遮挡
3. 街道反弓处不宜
4. 十字路口、丁字路口最佳
5. 气流平稳，忌直冲

### 3.3 店铺布局

**收银台**：
- 宜在财位
- 背后宜有靠（墙/柜）
- 忌背门

**货架摆放**：
- 高物在后，低物在前
- 动线流畅
- 忌中间堵死

## 四、都市风水小说写法

### 4.1 避免迷信嫌疑

**写法技巧**：
- 将风水解释为"空间能量学"
- 用"气流"、"气场"代替"灵气"
- 强调"心理暗示"作用
- 案例要有合理解释

### 4.2 专业术语使用

| 传统说法 | 都市小说写法 |
|:---|:---|
| 八卦 | 方位卦象 |
| 五行相生相克 | 能量流转规律 |
| 玄学 | 环境心理学 |
| 算命 | 趋势分析 |

### 4.3 主角能力设定

**推荐设定**：
- 精通八宅派（阳宅为主）
- 熟悉峦头派（形家）基础
- 了解玄空飞星（择日）
- 掌握奇门遁甲（高端）

**不建议**：
- 过于玄幻的能力
- 算命、看相（与主线无关）

### 4.4 剧情设计

**常见商业风水案**：
1. 商铺选址（街边店、商场店）
2. 公司办公室布局
3. 别墅/豪宅风水
4. 工厂/仓库风水
5. 阴宅（可选，争议较大）

**案例结构**：
1. 客户上门（商业合作）
2. 现场勘查（发现风水问题）
3. 分析原因（结合五行八卦）
4. 制定方案（专业建议）
5. 实施调整（具体操作）
6. 效果验证（生意变好）
