# 第58章 规格 (Spec)

## 📝 章节信息
- 章节编号: 58
- 标题: 绝境逃亡
- 前情回顾: 林夜用因果追溯救出沈暮雪，古墓崩塌，萧无痕重伤掩护，黑袍老者追击而出
- 目标字数: ≥4000字
- 时间线: 古墓外，崩塌后的混乱时刻

## 🎯 本章目标
1. 萧无痕与黑袍老者的师兄弟对决
2. 林夜带着沈暮雪逃跑的紧张场景
3. 揭示萧无痕的过去
4. 林夜的成长/决策

## 📖 剧情大纲

### 开端（0-25%）
- 林夜背着沈暮雪冲出烟尘
- 萧无痕挡在黑袍老者面前
- 紧张对峙

### 发展（25-50%）
- 萧无痕vs黑袍老者战斗
- 萧无痕暴露过去（师兄弟恩怨）
- 林夜犹豫是否逃走

### 高潮（50-75%）
- 萧无痕重伤，不敌黑袍老者
- 黑袍老者要夺舍林夜
- 林夜被迫应战/使用因果追溯

### 结尾（75-100%）
- 悬念式结尾
- 林夜做出关键选择
- 为第59章埋下伏笔
