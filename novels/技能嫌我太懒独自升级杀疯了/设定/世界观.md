# 世界观设定：《技能嫌我太懒，独自升级杀疯了》

## 一、作品定位

- **平台**：番茄小说（男频都市）
- **类型**：都市脑洞 + 反套路系统文
- **核心卖点**：躺平摆烂 + 技能自动升级 + 扮猪吃虎
- **受众**：都市男频读者，喜欢轻松爽文、反对内卷的上班族

## 二、核心设定

### 【金手指】万物自动升级系统

**触发条件**：主角越懒惰、越摸鱼、越摆烂

**升级规则**：
- 所有已掌握/接触过的技能，根据"摆烂时长、摸鱼程度、懒惰值"自动升级
- 技能等级无上限，可跨界融合进化
- 摆烂越狠，升级越快

**技能分类**：
| 类别 | 示例 | 说明 |
|:---|:---|:---|
| 生活技能 | 厨艺、驾驶、生活美学 | 日常自动觉醒 |
| 职业技能 | 办公、鉴宝、格斗、医术 | 接触即升级 |
| 特殊技能 | 精神异能、空间异能、自愈异能 | 顶级技能衍生 |

**奖励机制**：
- 技能每升一级，随机发放：现金返利、体质强化、稀有物品、隐藏异能
- 摆烂越久，奖励越丰厚

**禁忌（反套路核心）**：
- 主角一旦主动努力、内卷、刻意提升技能 → 系统暂停升级 + 惩罚性扣除懒惰值
- 彻底杜绝主角"努力"的动机，强化躺平核心

| 特殊 | 异能者、邪恶组织 | 后期都市异能圈势力 |

## 三、故事阶段

### 第一阶段：职场摆烂，技能初醒（1-100章）

**主线**：入职职场 → 激活系统 → 摸鱼打脸 → 辞职躺平

**核心爽点**：
- 连续摸鱼 → 技能自动升级 → 工作几分钟搞定
- 拒绝加班 → 被同事嘲笑 → 打脸反转
- 领导刁难 → 法律技能升级 → 反杀领导

**本阶段结束**：林辰辞职，获得第一桶金，开始宅家躺平

### 第二阶段：跨界升级，崭露头角（101-300章）

**主线**：宅家躺平 → 鉴宝/厨艺/赌石自动满级 → 行业打脸

**核心爽点**：
- 鉴宝技能LV100 → 一眼看穿真假 → 捡漏无数
- 厨艺技能 → 神级药膳 → 体质强化
- 格斗/医术自动满级 → 女主被吸引

**本阶段结束**：财富过亿，都市隐藏神豪

### 第三阶段：异能觉醒，横扫都市（301-600章）

**主线**：异能圈现世 → 系统衍生异能 → 轻松碾压

**核心爽点**：
- 顶级技能 → 衍生精神/空间/自愈异能
- 邪恶异能者挑衅 → 摆烂状态轻松反杀
- 成为异能圈无冕之王

### 第四阶段：全球布局，躺平封神（601-1000章+）

**主线**：全球旅居 → 每到一处技能自动满级 → 成为传说

**最终结局**：隐居世外桃源，财富无数，实力无敌，极致躺平

## 四、平台适配

| 维度 | 策略 |
|:---|:---|
| 标题 | 包含"技能"、"懒"、"升级"、"杀疯"等爆款关键词 |
| 节奏 | 每10章一个小爽点，30章一个大高潮 |
| 标签 | 都市、脑洞、系统、反套路、躺平、扮猪吃虎、神豪 |
| 更新 | 日更2-3章，维持读者粘性 |

## 五、禁忌事项（P0红线）

1. **主角不能主动努力** - 系统会惩罚
2. **不能有虐主情节** - 全程轻松爽文
3. **不能有复杂感情纠葛** - 甜宠为主，无狗血误会
4. **不能有明显AI词汇** - 保持文笔自然
