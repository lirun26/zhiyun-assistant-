## ⚠️ 小说创作规范（严格执行版）

### 7步创作流程
1. 读取skill指导
2. 章节spec规划
3. 创作正文
4. 评审报告
5. 修正问题
6. 更新记忆文件
7. 发送

### P0红线（绝对禁止）
- AI词汇/感悟式结尾/感叹式结尾/上帝视角
- 单章低于4000字
- 时间线矛盾

### P1红线（必须避免）
1. **重复描写**：同一事物/表达不能重复超过2次
2. **章节结尾雷同**：用grep检查前3章结尾关键词
3. **章节开头雷同**：检查地点/时间/人物状态是否重复
4. **中间整段重复**：grep整段关键词
5. **章节名与内容不符**：章节名必须反映核心事件
6. **情节与前文重复**：禁止"修炼→醒来→制定计划"固定模式
7. **修仙术语混乱**：都市类禁止修仙用语，修仙类禁止都市用语

### 评审检查清单
```
字数 ≥4000
P0检查：无AI词汇/感悟式结尾/感叹式结尾/上帝视角
开头对比：地点/时间/人物状态是否与前3章雷同
结尾对比：用grep检查"一步步走下去/清算的开始/等他xxx"等关键词
重复词：grep检查"微微一笑/淡淡的笑容/嘴角微微上扬"
修仙术语：grep检查"炼气/金丹/筑基/灵力/修为"等
一致性：主角姓氏/家族名称/时间线逻辑
```

### 术语替换规则
- "灵力" → "真气"
- "修为" → "道行"（高/低/恢复）
- "修炼" → "修炼道行"
