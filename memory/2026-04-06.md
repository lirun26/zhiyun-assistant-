# 2026-04-06 小说教训记录

## ⚠️ 重要错误：时间线矛盾

### 错误描述
第1章中"十年"与"三十天"矛盾：
- 原文："在末世里挣扎了十年，最后惨死"
- 但前文设定：极寒第30天降临，她被冻死

**正确理解：** 前世只活了30天（不到一个月）就死了，不是十年。"十年"是指上一世看到听到的别人的经历，不是她自己活了十年。

### 教训
写前世回忆时，必须明确区分：
- **她自己的经历**：末世爆发 → 极寒降临 → 被冻死（约30天）
- **她知道的别人的经历**：其他人可能活了几个月或几年

### 预防措施
1. 写前世回忆时，先检查第1章设定
2. 确认时间线一致性：爆发→极寒→死亡
3. 避免模糊时间词（十年/月/天）导致矛盾

# 2026-04-06 小说创作标准化 + OpenClaw升级

## 📝 小说创作流程标准化

### 学习并应用 open-novel-writing 标准流程
**标准流程：**
```
设定管理 → 大纲生成 → 章节规划(spec) → 正文生成 → 评审反馈
```

**第3章创作过程：**
1. ✅ 基于1-2章状态生成第3章规格（spec）
2. ✅ 按规格+写作五原则写正文（~3700字）
3. ✅ 评审：得分87.6/100（优秀，可发布）
4. ✅ 修复问题：金钱数字、消息来源、结尾表达
5. ✅ 发送到QQ邮箱

**写作五原则：**
- 声音：有立场的叙事者，不是摄像头
- 层次：情绪有起伏，不是单一色调
- 细节：动作具体化，有画面感
- 呼吸：长短句交替，张弛有度
- 活性：动词有力，少用"的"

**P0红线（绝对禁止）：**
- AI词汇（"众所周知"、"不言而喻"）
- 感悟式结尾（"他明白了..."）
- 感叹式结尾（"真是太..."）
- 上帝视角（"所有人没想到..."）
