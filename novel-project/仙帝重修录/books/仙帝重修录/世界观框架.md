# 《凡尘仙帝：重修混沌道》世界观框架

---

## 一、小说核心主题

**一句话卖点**：重生回到少年时代，带着前世仙帝记忆与残缺道基，从微末凡俗起步，逆天改命、补全道基、横扫仇敌、重登绝巅，这一世，不做无情仙尊，只守心中道，执掌自己的长生。

---

## 二、世界观设定

### 1. 世界格局

| 界面 | 描述 |
|------|------|
| **凡界** | 凡人王朝、宗门割据、灵气稀薄 |
| **灵界** | 大宗门、王朝圣地、真正修仙界 |
| **天界** | 仙帝、古神、道祖级存在的顶层世界 |
| **混沌界** | 禁忌之地，藏着世界本源与终极秘密 |

### 2. 核心势力

- 正道三宗：剑修、丹修、符修（表面光明，内部腐朽）
- 魔道四脉：杀伐、炼魂、毒修、血修（被世人误解）
- 凡界王朝：看似凡人，实则藏上古传承
- 上古世家：血脉传承，垄断资源
- 灵虚学院：大陆最著名的修仙学院，汇聚天才和强者
- 天庭势力：天界执法者，实则控制下界气运

### 3. 世界暗线（贯穿全书）

- 前世你并非正常陨落，而是被天道与天界联手算计
- 你的重生不是偶然，是世界意志自救
- 真正的敌人不是仇敌，是固化的等级秩序与伪天道

## 三、修炼体系

### 境界（每境分初期→中期→后期→圆满）

| 境界 | 描述 |
|------|------|
| 炼气境 | 引气入体 |
| 筑基境 | 筑道基 |
| 金丹境 | 结金丹 |
| 元婴境 | 化元婴 |
| 化神境 | 神念化形 |
| 炼虚境 | 虚空穿梭 |
| 合体境 | 身合天地 |
| 大乘境 | 渡劫前巅峰 |
| 渡劫境 | 引天劫 |
| 地仙→天仙→金仙→仙帝 | 仙人境界 |

### 核心设定：道基体系

- **道基分级**：凡基→灵基→道基→仙基→混沌道基
- **主角设定**：前世是混沌道基，被人打碎，重生后是残缺废基
- **金手指能力**：
  - 看穿别人道基缺陷
  - 修复、改造、升级道基
  - 自创道基，让废柴变天才

---

## 四、金手指

1. **重生回16岁** - 一切悲剧尚未发生
2. **前世仙帝级经验**：
   - 一眼看破功法破绽
   - 知道所有秘境、宝藏、机缘位置
   - 懂别人不懂的上古丹方、阵法、禁制
3. **残缺道瞳** - 能看穿灵脉、宝物、人心、功法本质
4. **无系统、无老爷爷** - 纯靠自己，逼格更高

---

## 五、法宝体系

### 1. 法宝等级

凡器→法器→宝器→灵器→仙器→神器→混沌至宝

### 2. 特色设定

- 本命道兵：与道基共生，越强越恐怖
- 记忆法宝：能储存前世经验、功法、剑意
- 界域类法宝：可随身携带小世界
- 因果类法宝：能斩断因果、规避死劫

### 3. 主角专属至宝

- **鸿蒙道卷** - 前世本命法宝残片，能记录大道、推演功法
- **光阴莲台** - 可短暂回溯时间、加速修炼
- **万化仙炉** - 能炼化万物，提纯灵气、炼制帝丹

## 六、爽点制造

### 1. 重生爽点
- 回到少年，救下前世惨死的亲人、爱人、兄弟
- 提前截胡本该属于仇敌的机缘、功法、美女

### 2. 扮猪吃虎爽点
- 表面废柴，实际仙帝
- 当众打脸嘲讽你的天才、长老、少主

### 3. 成长爽点
- 从凡界小城→横扫一域→威震灵界→征战天界
- 收小弟、立宗门、建王朝、成帝尊

### 4. 复仇爽点
- 前世欺你、叛你、杀你之人，一一清算
- 亲手打碎虚伪的天界秩序

### 5. 独特爽点（别人没有）
- 帮人修复道基，收获死忠
- 用凡火炼出仙丹，震惊全场
- 以凡人之躯，硬撼修仙大佬

## 七、剧情大纲（三卷结构）

### 第一卷：凡域崛起（1–100章）

- 重生归来，修复自身残缺道基
- 打脸家族反派、宗族长老
- 获得第一份机缘、第一柄剑、第一个小弟
- 走出小城，参加宗门大比，一鸣惊人
- 建立自己的势力，收拢人心

### 第二卷：灵界称尊（501–1500章）

- 进入真正修仙界
- 揭露宗门阴谋，颠覆腐朽秩序
- 寻找前世碎片，恢复部分实力
- 建立无上宗门，成为一界之主
- 揭开前世陨落真相

### 第三卷：天界争霸（1501–3000章）

- 飞升天界，直面最终仇敌
- 重组道基，成就混沌道体
- 对抗伪天道，打破天地枷锁
- 最终证道，成为真正的自由仙帝
