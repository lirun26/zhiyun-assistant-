## 🎯 二、游戏玩法

### 2.1 核心规则

```
1. 牌堆中找出两个相同的麻将牌
2. 点击第一个，再点击第二个
3. 两者即可消除
4. 消除所有牌即胜利
```

### 2.2 牌型设计

| 牌型 | 数量 | 图案 |
|------|------||
| 万子 | 9种 × 4张 = 36张 | 🀇🀈🀉🀊🀋🀌🀍🀎🀏 |
| 筒子 | 9种 × 4张 = 36张 | 🀙🀚🀛🀜🀝🀞🀟🀠🀡 |
| 条子 | 9种 × 4张 = 36张 | 🀐🀑🀒🀓🀔🀕🀖🀗🀘 |
| 东南西北中 | 7种 × 4张 = 28张 | 🀀🀁🀂🀃🀄🀅🀆 |
| 春夏秋冬梅兰竹菊 | 8种 × 1张 = 8张 | 🀓🀔🀕🀖🀗🀘🀙🀚 |

**总计**：144张牌（36对）

### 2.3 难度递进

| 关卡 | 牌数 | 限时 | 特色 |
|------|------|------|------|
| 1-10 | 36对 | 无 | 简单 |
| 11-30 | 50对 | 60秒 | 中等 |
| 31-50 | 60对 | 45秒 | 困难 |
| 51+ | 72对 | 30秒 | 专家 |

### 2.4 道具系统

| 道具 | 功能 | 价格 |
|------|------|------|
| 💣 炸弹 | 消除任意3张牌 | 20💰 |
| 🔍 提示 | 高亮显示一对 | 15💰 |
| 🔄 重置 | 打乱当前牌序 | 30💰 |
| ⏰ 加时 | 增加30秒 | 25💰 |

## 🏗️ 三、系统架构

### 3.1 页面结构

```
首页
├── 开始游戏 → 关卡选择 → 游戏页 → 胜利/失败
├── 每日挑战 → 每日3关
├── 道具商店 → 购买道具
└── 排行榜 → 好友排行
```

### 3.2 数据结构

```javascript
// 关卡数据
Level {
  id: number,
  pairs: number,      // 牌对数
  time: number,      // 限时(秒)，0=无限
  hint: boolean,     // 是否显示提示
  special: string    // 特殊规则
}

// 用户数据
User {
  level: number,     // 当前关卡
  coins: number,     // 金币
  items: {          // 道具数量
    bomb: number,
    hint: number,
    reshuffle: number,
    time: number
  }
}
```
